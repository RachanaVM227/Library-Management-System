# 📚 Library Management System (LMS)

A modern full-stack **Library Management System** designed for colleges and universities with dedicated portals for **Users, Employees, and Admins**.

---

# 🚀 Features

## 👨‍🎓 User Portal
- Search books
- View available books
- Book details modal
- Read blogs & announcements
- Responsive dashboard

## 👨‍💼 Employee Portal
- Add books
- Update books
- Delete books
- Manage inventory
- Track top 5 added books

## 👨‍💻 Admin Portal
- Manage students & employees
- System analytics
- Monthly reports
- Histogram chart for popular books
